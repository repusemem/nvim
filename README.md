# nvim

## Introduction

This is my dotfiles of lazy-nvim, which has some vital plugins to turn nvim into an IDE:
- catpuccin: theme to make its colorful
- codex: codex for anyone wants to use AI
- nvim-ufo: collapsing code block to make life easier
- rainbow-delimeter: making brackets more colorful
- toggleterm: open terminal while in nvim
- and a lot more...

## Installation

1. Download it
2. Put it in the ~/.config/nvim folder
3. Open nvim
4. Wait for it to load and install `:Lazy sync`
5. Enjoy it!
